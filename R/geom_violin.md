# Quick Start

```r
library(ggplot2)

p <- ggplot(mtcars, aes(factor(cyl), mpg))
p + geom_violin()+ geom_point() + theme_light()
```

<a name="dw7gw"></a>
## scale
```r
p + geom_violin(scale = "width")+ theme_light()
```

## draw_quantiles
```r
p + geom_violin(aes(fill = factor(cyl)),draw_quantiles = c(0.25, 0.5, 0.75))
```
