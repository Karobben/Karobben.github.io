Regression in R

linear equation with one unknown

test data: mpg

Quick start

Call:
lm(formula = hwy ~ displ, data = mpg)

Coefficients:
(Intercept)        displ  
     35.698       -3.531

As we can see above, the function is

$$
y = -3.531x+35.698
$$

y=ax^2+bx+c
