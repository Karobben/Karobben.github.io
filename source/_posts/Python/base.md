### Float formate
```python
##########print##########

pi = 3.141592653
print('%10.3f' % pi) #字段宽10，精度3
## 3.142
print("pi = %.*f" % (3,pi)) #用*从后面的元组中读取字段宽度或精度
## pi = 3.142
print('%010.3f' % pi) #用0填充空白
## 000003.142
print('%-10.3f' % pi) #左对齐
## 3.142
print('%+f' % pi) #显示正负号
## +3.141593
```
