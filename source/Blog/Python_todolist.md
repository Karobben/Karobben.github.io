# 3 放入列表

写一个循环读取, 并放入columns, columns 叠入Pile
```python
def TODO_col(TODO):
    Column_list = []
    for Sub in TODO:
        Sub_list = []
        for i in Sub:
            Sub_list += [urwid.Text(i)]
        Column_list += [urwid.Columns(Sub_list)]
    A = urwid.Pile(Column_list)
    return A
```
