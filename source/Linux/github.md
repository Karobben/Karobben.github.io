# github


Github 本地上更新库
```bash
# Initialize your directory
git init

# 关联
git remote add IO https://github.com/Karobben/Karobben.github.io

#添加
git add .

#注释
git commit -m "注释"

git pull --rebase IO master
git push -u IO master
```
