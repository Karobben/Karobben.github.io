# GIMP



# Line
## Straight line

1. `鼠标左键` 选择起始位置
2. 按住`shift`， 锁定
3. 松开`鼠标左键`， 不松`shift`
4. 移动到终点，点击`鼠标左键`


## Rectangle
1. Rectangle select tools (select a region)
2. Edit -> Stroke selection -> Stroke
