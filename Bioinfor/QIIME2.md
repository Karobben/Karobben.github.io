# QIIME2

# 還沒更新
```
# 激活工作环境，需要几十秒
source activate qiime2-2017.6

# 检查是否安装成功，弹出程序帮助即成功
qiime --help

# 关闭工作环境：不用时关闭，不然你其它程序可能会出错
source deactivate
```
