How to install scrcpy
in Deepin 15.11


Karobben
5 Feb 2020



Scrcpy is an powful open source crassplatform softerware which could projet your android phone to the computer. And it is interactable, it means you can control your phone with keyboard and mouse (You can imput english only). Scrcpy is extreamly easy to install at Ubuntu with the help of apt, but it is unavallable in deepin. Howevery, you can still using "snap" to install it as a pice of cack, in some region only. Because in other place like China, snap install are too slow to use. So, groups of people may inavitablly to build scrcpy by source code. It is easy for the people who familiar with mason and nijia, but it is a nightmare for people like me, who even hard to reading the error codes. So, I spend few hourse to figure out how to build this app in deepin 15.11 and to help the people like me to save the day.

I tried to install the latest version(1.12.1), but I have no idea why it ask me for "scrcpy-server-v1.11.jar". So, I tried building V1.12.1 with scrcpy-server-v1.11.jar, It works in meson and nijia when you conigure and build scrcpy, and when you run the scrcpy, everything just looks like fine, terminal resbonded with correct codes, and my phone responded it. But there is no window jump out to the screen. Creepy wired. So, I deleted all packages and installed the version V1.11.1, and boom, it finally works! Cheer!

1 Set up your environment


Terminal 

sudo apt install adb ffmpeg libsdl2-2.0.0 make gcc pkg-config meson ninja-build \
  libavcodec-dev libavformat-dev libavutil-dev libsdl2-dev

If apt is running slow, please go to switch an aporpriate source mirror
An example of how to change your source:

Terminal 

## Generated by deepin-installer
deb [by-hash=force] http://ftp.sjtu.edu.cn/deepin panda main contrib non-free
#deb-src http://ftp.sjtu.edu.cn/deepin panda main contrib non-free




2 Dowloading Source Packages


Releas location: Genymobile Github Download the files in red boxs. In this section, we need to add ".jar" to the tail of scrcpy-server-v1.11 and install it later. And this is why I can't build it correctly at first two hours.

Terminal 

wget -c https://github.com/Genymobile/scrcpy/releases/download/v1.11/scrcpy-server-v1.11
wget -c https://github.com/Genymobile/scrcpy/archive/v1.11.tar.gz

mv scrcpy-server-v1.11 scrcpy-server-v1.11.jar
tar -zxvf v1.11.tar.gz




3 Install

Install the server

Terminal 

sudo install scrcpy-server-v1.11.jar /usr/local/bin/scrcpy-server.jar

Configure and Build

Terminal 

cd scrcpy
meson build --buildtype release --strip -Db_lto=true  -Dprebuilt_server=../scrcpy-server-v1.11.jar
cd build
ninja
sudo ninja install

You might receve this error

Terminal 

ninja: error: '/home/ken/Soft/scrcpy-server-v1.11.jar', 
needed by 'server/scrcpy-server', missing and no known rule to make it

You can fix it by cp or move scrcpy-server-v1.11.jar to '/home/ken/Soft/scrcpy-server-v1.11.jar', in this example.




4 Porject your Androird phone

Connect your Androird Phone via USB, and open USB developer mode and make sure you enable USB debugging on your Android device.

mv scrcpy-server-v1.11 scrcpy-server-v1.11.jar
